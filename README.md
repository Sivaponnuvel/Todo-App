# 📝 Todo Manager

A modern and responsive Todo Manager application built using React.js. This project helps users efficiently manage daily tasks with features like adding, editing, deleting, and tracking task completion.

---

## 🚀 Features

* ➕ Add New Tasks
* ✏️ Edit Existing Tasks
* 🗑️ Delete Tasks
* ✅ Mark Tasks as Complete / Incomplete
* 📊 Task Statistics (Total & Completed Tasks)
* 🧹 Clear All Tasks
* ⌨️ Add Tasks Using Enter Key
* 📱 Responsive and User-Friendly Interface

---

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* CSS3
* Vite

---

## 🎨 Features Overview

### 📝 Task Management

Users can:

* Add new tasks
* Edit existing tasks
* Delete unwanted tasks

### ✅ Task Completion

* Mark tasks as completed
* Toggle completion status anytime
* Completed tasks are visually highlighted

### 📊 Task Statistics

Displays:

* Total number of tasks
* Number of completed tasks

### 🧹 Clear All Tasks

* Remove all tasks with a single click

### 🎯 User Experience

* Prevents empty task submission
* Supports adding tasks using the Enter key
* Clean and modern interface

## 💡 How to Run

Clone the repository:
```bash
git clone https://github.com/Sivaponnuvel/Todo-App
```

Open the project folder:
```bash
cd todo-app
```

Install dependencies:
```bash
npm install
```

Start the development server:
```bash
npm run dev
```

Open the URL shown in the terminal (usually http://localhost:5173).
